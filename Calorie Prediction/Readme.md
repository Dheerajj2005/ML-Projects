# Calorie Expenditure Predictor

This project predicts calories burned during exercise using a supervised regression pipeline built with `scikit-learn` and deployed via `Streamlit`.

## Project components

- `exercise.csv`, `calories.csv`: raw dataset files
- `calorie-prediction.ipynb`: exploratory analysis, model comparison, final training and model export
- `app.py`: Streamlit interface for live prediction using saved model
- `Calorie_model.pkl`: trained model (XGBRegressor)

## Data schema

Input features:
- `Gender` (male/female; encoded `0/1`)
- `Age`
- `Height`
- `Weight`
- `Duration`
- `Heart_Rate`
- `Body_Temp`

Target:
- `Calories`

## Data preprocessing

- dropped `User_ID` (irrelevant)
- converted `Gender` to numeric
- standard-scaled features before prediction in `app.py`

## Model workflow (notebook)

1. train/test split (80/20)
2. baseline regression with `LinearRegression`
3. grid search over:
   - `LinearRegression`
   - `Lasso`
   - `Ridge`
   - `ElasticNet`
   - `DecisionTreeRegressor`
   - `XGBRegressor`
4. polynomial features with `LinearRegression`
5. final model: `XGBRegressor` (best test performance)
6. `pickle.dump(model, "Calorie_model.pkl")`

## Streamlit app (`app.py`)

- loads `exercise.csv` + `calories.csv` and builds UI sliders
- applies same gender encoding as training
- scales features with `StandardScaler`
- loads `Calorie_model.pkl`
- predicts calories when button clicked

### prediction output
- returns rounded value as `int(round(y_pred * 100))` (same logic as existing app)

## Quick run

1. install
   ```bash
   pip install pandas numpy scikit-learn xgboost streamlit
   ```
2. run app
   ```bash
   streamlit run app.py
   ```

## Notes

- the model is regression, and final selection is XGB due to highest score in notebook.
- ensure `Calorie_model.pkl` exists (generated by `calorie-prediction.ipynb`).
- because it uses scaling, inference must apply same feature transformation.
